Personal information & Add-on's permissions

This add-on doesn't process or store any personal information about google accounts of the users.

During the authorization process users of the add-on have to grant access to the following scopes: access to the following google api scopes:

  • https://www.googleapis.com/auth/script.container.ui - is needed to open the sidebar in the context of user's Google Spreadsheet with add-on's user interfaces;
  • https://www.googleapis.com/auth/script.external_request - is needed for making api calls to the 3rd party services, which are apaleo open API Platform in the current case;
  • https://www.googleapis.com/auth/spreadsheets - is required for populating user's Spreadsheet with the aggregated data, received for apaleo Platform.

Connection to apaleo Platform

apaleo Google Sheet Add-on requires an account in apaleo Platform. It means that the data transferred to apaleo for authentication and authorisation purposes together with the data received form apaleo API to build a report are secured and protected by apaleo's Privacy Policy
