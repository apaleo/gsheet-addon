Personal Information & Add-on Permissions

This add-on doesn't process or store any personal information about google accounts of the users.

During the authorization process users of the add-on have to grant access to the following Google API scopes:

  • https://www.googleapis.com/auth/script.container.ui - is needed to open the sidebar in the context of a user's Google Sheet with add-on's user interfaces;
  • https://www.googleapis.com/auth/script.external_request - is needed for making API calls to the 3rd party services, which in the current case are apaleo API-s;
  • https://www.googleapis.com/auth/spreadsheets - is required for populating user's Google Sheet with aggregated data received from the apaleo Platform.

Connection to apaleo Platform

apaleo for Sheets requires an account in the apaleo Platform. It means that the data is transferred to apaleo for authentication and authorization purposes, together with the data received from the apaleo API to build a report, are secured and protected by apaleo's Privacy Policy
